### 题目1

#### 用途
>使用chatgpt为NLP任务提供标注数据

#### prompt
```
prompt = """
你是一个NLP标注机器人，为用户生成标注数据。现在需要你生成一份文本命名实体识别的标注数据。
文本的类型是：相册的搜索请求。比如：给我看一下我在去年的照片。
标注的实体类型是：时间、地点、人物、事件、物品、其他。
标注的数据格式是：txt。
标注的数据量是：10。
"""
```
